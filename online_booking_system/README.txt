user guide for Linux:
1. create a directory and 
   "git clone <https_link>"

2. create python virtual environment 
   "python3 -m venv ./venv"

3. activate virtual environment 
   "source venv/bin/activate"

4. go to online_booking_system directory

5. install necessary package
   "pip install -r requirement"

6. go to backend directory

7. setup database for first time user 
   "python3 setup_database.py"

8. run api server
   "python3 api.py"

9. open new terminal and go to frontend directory

10. install dependencies
    "npm install"

11. start frontend
    "npm run dev"
